### 关于我 / About Me

> 一名在澳门读书的高中生。白天上课，晚上写代码。
>
> I'm a high school student in Macau. Classes by day, coding by night.

最开始接触编程是因为喜欢 Minecraft，慢慢地从开服、调参开始，写起了一些小工具。后来遇见了 Koishi，发现机器人比想象中有趣得多，于是也试着把自己的想法做成插件。大多数项目都是在深夜调试出来的，安静的时段里写代码对我来说是一种放松。

I first got into programming because of Minecraft. It started with running servers and tweaking parameters, which led to writing small utilities. Later I discovered Koishi and found bots far more interesting than I expected, so I started turning my own ideas into plugins. Most of my projects are debugged late at night — coding in the quiet hours is a way for me to unwind.

技术栈主要围绕着 TypeScript 和 Node.js，偶尔也会用到 Python 和前端的东西。

My tech stack revolves around TypeScript and Node.js, with occasional use of Python and front-end tools.

如果你对我的仓库感兴趣，随便翻翻就好。能帮到你的话，我会很高兴。

If any of my repositories catch your interest, feel free to look around. I'd be glad if something here is useful to you.
